## Featured repositories
- [adbsmsforward-openwrt](https://github.com/edikurexe/adbsmsforward-openwrt) — Forward SMS from Android devices via ADB to Telegram for lightweight OpenWrt-based workflows.
- [satpam-bot-telegram](https://github.com/edikurexe/satpam-bot-telegram) — Telegram moderation and monitoring bot for group management and username rule enforcement.
- [build-ipk-openwrt](https://github.com/edikurexe/build-ipk-openwrt) — GitHub Actions workflow for building OpenWrt IPK packages from external sources.
- [ULO](https://github.com/edikurexe/ULO) — OpenWrt builder workflow for multiple devices and SoC targets.
- [extract-img](https://github.com/edikurexe/extract-img) — Automated utility for extracting Linux and OpenWrt rootfs images using GitHub Actions.
